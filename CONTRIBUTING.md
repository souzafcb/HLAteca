# Como contribuir

## Tipos de contribuição
- Inclusão de documentos históricos
- Correção de metadados
- Organização estrutural
- Scripts de automação

## Regras
- NÃO subir arquivos de áreas restritas/login
- Sempre incluir metadados
- Nomear arquivos conforme padrão

## Nomeação padrão
fabricante_produto_tipo_versao_data.ext

Exemplo:
onelambda_labtype_drb345_datasheet_v2-1_2019.pdf

## Metadados obrigatórios
- fabricante
- produto
- tipo de documento
- versão
- data (se conhecida)
- fonte (URL ou origem)

## Processo
1. Fork
2. Commit
3. Pull request
